# Indian Diseases Analysis

## Overview

Indian Diseases Analysis is a Data Analytics project that focuses on exploring healthcare data to identify disease patterns, treatment costs, recovery trends, and patient demographics. The project uses Python-based data analysis and visualization techniques to transform raw healthcare data into meaningful insights.

## Objectives

* Analyse disease prevalence across different patient groups.
* Study disease severity and recovery trends.
* Evaluate treatment costs associated with various diseases.
* Identify relationships between lifestyle factors and disease occurrence.
* Generate visual insights for better healthcare decision-making.

## Dataset

Dataset: Indian Patient Disease & Treatment Dataset

Source:
https://www.kaggle.com/datasets/ashyou09/indian-patient-disease-and-treatment-dataset

### Dataset Features

* Patient Demographics
* Disease Category
* Disease Severity
* Treatment Cost
* Recovery Days
* Smoking Status
* Alcohol Consumption
* BMI
* Comorbidity Information
* State and Region Information
